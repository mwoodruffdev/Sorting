# SortMeOut

SortMeOut is a simple app to allow students and developers to learn and understand basic fundamental Computer Science Algorithms and concepts.

Presently, SortMeOut supports four basic Sorting Algorithms:

- Bubble Sort
- Merge Sort
- Quick Sort
- Insertion Sort
